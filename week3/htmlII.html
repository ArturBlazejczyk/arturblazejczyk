Atrybuty (i nowe znaczniki)

Nowe znaczniki

  • < article> - samodzielna treść
  • < b> & < i> - nie mają dużego znaczenia semantycznego, służą do wyróżniania fragmentu z tekstu otaczającego
  • < strong> - podkreśla wysoką istotność i poważność frazy w nim zawartej (pogrubienie)
  • < em> - kładzie mocniejszy nacisk na zawartość (kursywa)
  • < a href="" target=""> opis łącza < /a> - linki/łącza wewnątrz dokumentu
  • < element id="moj-identyfikator"> - atrybut id - identyfikuje element (przydaje się do nawigacji w obrębie elementu)
  • < img src="" alt=""> - obrazek o podanym adresie

Podsumowanie

  • atrybuty są "niewidzialne" - chowamy je w znaczniku
  • nadają elementom dodatkowe właściwości (href, src)
  • pozwalają na grupowanie elementów (class)
  • pozwalają na identyfikowanie elementów (id)
  • można je miksować
  • spis atrybutów "globalnych"
