JS = JavaScript

Pliki zawierające kod JavaScript mają rozszerzenie .js

Będziemy uczyć się ECMAScript 2015 (ES6)

Co to jest ECMAScript 6(+)

  • ECMAScript jest to ustandaryzowany przez organizację ECMA skryptowy język programowania
  • JavaScript implementuje rzeczy ze specyfikacji ECMAScript
  • Rozwój ECMAScript mocno przyspieszył przez szerokie wykorzystywanie JavaScriptu
  • ES6 (i nowsze) wprowadzają wiele udogodnień dla developerów
  • Nowe feature'y mogą być niewspierane we wszystkich przeglądarkach (ale są na to sposoby 😎)

Jak działa JavaScript?

  • console.log('Siema!'); - wyświetla wiadomość 'Siema!' w konsoli

    • Zmienne w JS

    • const - rzeczy mające stałą strukturę, nie można ich podmienić
    • let - rzeczy, które możemy przypisywać wielokrotnie, ale definiujemy tylko raz!

      • Zmienne zamieszczamy dodając przed nimi znak '$' np.

    • console.log('Nazywam się ${name} i mam ${age} lat')

    Prawie wszystko w JS jest obiektem.

    Typy proste w JS

    • String - tekst, otaczamy ' lub "
    • Number - liczba
    • Bigint - duża liczba
    • boolean - true lub false
    • null - wartość celowo niezdefiniowana
    • undefined - brak definicji
    • symbol - tworzy unikatowe opakowanie

    Podsumowanie

    • Kod JS uruchamia się raz (po załadowaniu strony)
    • Java to nie JavaScript
    • JavaScript implementuje rzeczy ze specyfikacji ECMAScript
    • JavaScript to frontendowe programowanie
    • Za pomocą JS można robić różne szalone rzeczy
