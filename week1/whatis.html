Czym jest frontend?

Powrót do strony głównej

Frontend

  • odpowiada za wygląd stron / aplikacji internetowych
  • odpowiada za interakcje z użytkownikiem
  • jest generowany głównie przy użyciu trzech technologii: HTML, CSS i JavaScript (istnieje jeszcze m.in. WebAssembly, jednak nie jest jeszcze super popularne)
  • technologie frontendowe przejmują rynek tradycyjnych aplikacji (w wybranych obszarach)
