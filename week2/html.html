HTML

Powrót do strony głównej

HTML = HyperText Markup Language

HTML odpowiada za treść strony (a właściwie za jej właściwe opakowanie)

Treść to:

- Teksty
- Nagłówki
- Linki
- Listy
- Definicje
- Obrazki
- Video
- Formularze
- Wszystko co jest nośnikiem informacji

HTML składa się ze znaczników/tagów/elementów

Znaczniki

main - główna część dokumentu
section - sekcja dokumentu
header - nagłówek dokumentu/sekcji
footer - stopka dokumentu/sekcji
h1-h6 - tekstowe nagłówki tytułowe
p - paragraf

HTML

- Opakowuje i opisuje treść strony
- Mówi przeglądarce o strukturze dokumentu
(i o tym co ma wczytać) - Posiada swoją zdefiniowaną składnie
- Nie jest językiem programowania, jest językiem znaczników (Markup Language)
- Aktualną rekomendowaną wersją jest 5.2
- Całą specyfikację znajdziesz na stronie w3c
